# Scharp
## By Misrocoft
*"It's C#, but dyslexic."*

### Installation (Ubuntu)
```bash
sudo make install
```

### Usage
#### Create a new scharp console app
```
netdot new console -o <project name>
```

### Compile and run
```
netdot run
```

### Uninstall (idk why you would ever want to tho)
```bash
sudo make uninstall
```
